## Getting Started

This project was used in version Python 3.9+

### Setup
```
$ python3 -m venv venv 
$ source .venv/bin/activate
```

### Install Dependencies
```
$ pip install -r requirements.txt
```

### Startup
```
$ python3 application.py
```
